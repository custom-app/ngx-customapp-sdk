request

  • request(endpoint: string, req?: RequestType, headers?: HttpHeaders, disableAuth?: boolean): Observable<ResponseType>
    • Defined in angular-packages/projects/ngx-customapp-proto-http/src/lib/services/request.service.ts:81

    Makes an http request.

    Example:

    @Injectable()
    export class MenuService {

      constructor(
        private request: RequestService<Request.AsObject, Response.AsObject>,
      ) {}

      getCategoryList(): Observable<CategoryListResponse.AsObject> {
        return this.request.request(
        baseUrl + '/category/list',
        {categories: {}} // instance of the Request.AsObject
        ).pipe(map(resp => resp.categories!))
      }
    }

    Parameters

    • endpoint: string

      A destination for the request.

    • Optional req: RequestType

      The request body. Passed to the ProtoHttpConfig.serializer before sending the request.

    • Optional headers: HttpHeaders

      Some headers. For auth header it is recommended to use AuthInterceptor from ngx-customapp-jwt.

    • Optional disableAuth: boolean

      Disables AuthInterceptor from ngx-customapp-jwt.

    Returns Observable<ResponseType>
